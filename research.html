Data & Data Management

As Data Manager for the Coastal Critical Zone group, I work with our researchers to manage their datasets using the FAIR principles. Visit my user page on HydroShare to see discoverable datasets. Use the ketword tags #CCZN and #CZNet to find all the datasets associated with our Coastal group.

Find examples of the Python codes we use to process and correct hydrological sensor data and write deployment metadata on GitHub.

Codes to manage and validate water quality data can be found here.

Visualize some data from the Coastal Critical Zone Network monitoring site

This data is depth to water (from ground surface) in shallow monitoring wells at several Coastal Critical Zone (#CCZN) sites across the Delmarva peninsula. Metadata and dataset details are described on HydroShare.
